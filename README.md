##Simple eCommerce Front-End Template

A simple eCommerce front-end website template that showcases products with a clean and modern layout. It includes essential pages such as a homepage, shop, categories, about, and contact page. The project is designed as a basic starting point for building an online store interface.

#Features

Clean and modern UI design

Product listing layout

Categories section

Shop page with basic filtering UI

About and Contact pages

Responsive design for desktop and mobile devices

Built using pure HTML, CSS, and JavaScript (no frameworks)


#Technologies Used

HTML5

CSS3

JavaScript

MySQL

#Purpose

This project serves as a basic front-end template for learning or starting an eCommerce website. It focuses on layout, structure, and responsive design without backend functionality.
